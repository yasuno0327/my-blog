# Shokubunka media project

## Overview
This is media api sample written in go.

Use
Golang (gin)
Docker

## Setup

### 1 Install Docker
[Docker for Mac](https://store.docker.com/editions/community/docker-ce-desktop-mac)

### 2 Clone frontend
Under development

### 3 Build image
Run next command

```shell
docker-compose build
```

### 4 Run app

```shell
docker-compose up
```
